# Bus Seat Manager

Sistema de gerenciamento de assentos de ônibus desenvolvido com React + Vite + TypeScript.

## Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Zustand (gerenciamento de estado)
- Axios (requisições HTTP)
- MSW (Mock Service Worker)
- Lucide React (ícones)

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Funcionalidades

- **Dashboard**: Visão geral com estatísticas e próximas viagens
- **Gerenciamento de Ônibus**: CRUD completo de ônibus com configuração de layout de assentos
- **Gerenciamento de Viagens**: CRUD completo de viagens
- **Gerenciamento de Passageiros**: CRUD completo de passageiros com busca
- **Mapa de Assentos**: Visualização interativa do mapa de assentos da viagem com:
  - Atribuição de passageiros a assentos
  - Liberação de assentos
  - Bloqueio de assentos
  - Visualização de status (livre, ocupado, bloqueado)
  - Tooltip com informações do passageiro
